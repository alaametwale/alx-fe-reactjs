# ALX React App

This project is part of the ALX Front-End Web Development program.

## Description
This is a basic React application created using Vite.  
It is used to learn the fundamentals and core concepts of React.

## Technologies Used
- React
- JavaScript
- Vite
- Node.js

## How to Run the Project
1. Install dependencies:
   npm install
2. Start the development server:
   npm run dev
3. Open your browser and go to:
   http://localhost:5173
